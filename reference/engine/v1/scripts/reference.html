Script structure

  • name - string
  • description - string (optional)
  • modules - [Module]
  • functions - Functions Object
  • steps - [Step]
  • cleanup - [Step]
    Cleanup tasks are always run, regardless of whether any previous steps or cleanup tasks failed.

Module

Functions Object

Object containing custom functions. Function names are provided as the property key, with the value being a Function

Function

Inline JavaScript. Can be defined as a block of code, or an array containing parameter names followed by the code. functions: singleLineFunc: return { message: "this is a single line of code " } multiLineFunc: > return { message: "this spans a few lines" } funcWithArgs: - name - age - return "User " + name + " is " + age + " years old!"
